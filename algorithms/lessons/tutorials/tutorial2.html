什么是变量？

想象一下，你有一个神奇的盒子，这个盒子可以装下你想要的任何东西：数字、文字、颜色，甚至是一颗小星星。更棒的是，你可以给这个盒子起一个名字，这样你每次需要它的时候，只要叫出它的名字，就能轻松找到它！在编程中，这个“神奇的盒子”就是变量。

变量是程序中用来存储数据的容器。你可以把信息放进去，然后随时取出来用。在 JavaScript 中，变量可以存储很多不同类型的数据，例如数字、文字（字符串）、布尔值（true 或 false）等。

让我们来看一个简单的例子：

var stars = 1;

在这行代码中，我们创建了一个叫做 stars 的变量，并且把数字 1 存进了这个“盒子”里。这意味着，现在这个 stars 变量就代表了数字 1。

联系上文：理解 stars = 1

现在，我们来看看右边代码编辑器中的这行代码：

var stars = 1;

你已经知道，这行代码创建了一个名为 stars 的变量，并且给它赋值为 1。但这并不仅仅是一个数字，它实际上还控制着页面上星星的数量！目前，stars = 1 代表页面上将显示一颗星星。

动手试试：增加星星的数量

现在轮到你了！让我们通过修改变量的值，来改变页面上的星星数量。你只需要将 stars = 1 这一行代码修改为 stars = 3，然后点击“运行”按钮。

  1. 在代码编辑器中找到 stars = 1; 这行代码。
  2. 将 1 改成 3，就像这样：
var stars = 3;

点击“运行”按钮，看一看页面上会发生什么变化！

你会发现，页面上现在出现了三颗星星！这就是变量的魔力——通过简单的修改，你可以轻松地控制和调整程序的行为。
