第五课：循环魔法阵的奥秘

欢迎来到第五课！

在这节课中，我们将探讨一种非常强大的工具——循环语句。就像在魔法阵中点亮一颗颗星星一样，循环语句可以让我们轻松地重复执行一段代码，从而实现复杂的图形效果。

什么是循环语句？

循环语句是编程中的一种控制结构，它可以让一段代码反复执行，直到满足特定的条件。这种机制让我们不必手动重复编写代码，而是通过简单的语句让计算机帮我们完成重复的工作。

for 循环语句

for 循环是最常见的循环语句之一，它的结构如下：


for (初始化; 条件; 增量) {
    // 这里的代码会被重复执行
}

让我们来逐一解释：

  • 初始化： 在循环开始前执行一次，用于定义循环变量。例如 let i = 0;。
  • 条件： 每次循环开始时都会检查这个条件。如果条件为真，循环继续；如果为假，循环结束。例如 i < 6;。
  • 增量： 每次循环结束后执行，用于更新循环变量。例如 i++;，表示 i 的值每次增加 1。

综合起来，for 循环的完整流程如下：

  1. 执行初始化部分。
  2. 检查条件。如果为真，进入循环体执行代码；否则跳出循环。
  3. 执行循环体的代码。
  4. 执行增量部分，然后回到第 2 步，继续循环。

让我们来看一个具体的例子：


for (let i = 0; i < 6; i++) {
    console.log(i);
}

在这个例子中，i 从 0 开始，每次循环加 1，直到 i 达到 6 为止。控制台会输出 0 到 5 的数字。

while 循环语句

while 循环也是一种常见的循环语句，它的结构如下：


while (条件) {
    // 只要条件为真，这里的代码就会被重复执行
}

让我们来详细解释：

  • 条件： 在每次循环之前都会检查这个条件。如果条件为真，循环继续；如果为假，循环结束。

与 for 循环不同的是，while 循环在循环体内通常会有更新循环变量的代码，否则可能会进入无限循环。例如：


let i = 0;
while (i < 6) {
    console.log(i);
    i++;
}

这个例子与前面的 for 循环效果相同，会输出 0 到 5 的数字。
