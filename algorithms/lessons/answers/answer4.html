完成任务：下面是完整的条件语句代码，根据天气状况来决定绘制的内容。


function drawSky() {
    ctx.clearRect(0, 0, 500, 500);
    if (isDaytime) {
        drawSun(); // 绘制太阳
        if (isClearSky) {
            drawHappyChild(); // 绘制一个快乐的小朋友
        }
    } else {
        drawMoonAndStars(); // 绘制月亮和星星
        if (!isClearSky) {
            drawSadChild(); // 绘制一个不快乐的小朋友
        }
    }
}
drawSky(); // 运行函数以应用条件

这段代码通过检查 isDaytime 和 isClearSky 的值来决定绘制哪种天气情景。
