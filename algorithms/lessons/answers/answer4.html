完成任务：下面是完整的条件语句代码，根据天气状况来决定绘制的内容。

function drawSky() {
    if (isDaytime) {
        drawSun(); // ☀️ 绘制太阳
        if (isClearSky) {
            drawHappyChild(); // 😊 绘制一个快乐的小朋友
        }
    } else {
        drawMoonAndStar(); // 🌙 🌟 绘制月亮和星星
        if (isClearSky == false) {
            drawDimStar(); // 🌑 绘制暗淡的星星
            drawHiddenMoon(); // ☁️ 月亮被云遮住
            drawSadChild(); // 😞 绘制一个不快乐的小朋友
        }
    }
}
drawSky();

这段代码通过检查 isDaytime 和 isClearSky 的值来决定绘制哪种天气情景。
