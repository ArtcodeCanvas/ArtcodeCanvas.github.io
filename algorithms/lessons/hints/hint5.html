第五课：循环魔法阵 - 提示

任务回顾：使用循环语句点亮魔法阵中的所有星星。

如何使用循环语句：

  • for 循环：通常用于已知执行次数的情况。格式如下： 
    for (初始化; 条件; 增量) {
    // 循环体代码
}
  • while 循环：适用于需要反复执行，直到某个条件不再满足为止。格式如下： 
    while (条件) {
    // 循环体代码
}
  • 在这次任务中，你需要修改 for 循环的条件部分，使得循环能够执行六次。
  • 每次循环，都会调用 changeStarToMoon() 函数来点亮一颗星星。

提示：确保你了解 for 循环的结构，并将其应用到代码中以完成任务。
