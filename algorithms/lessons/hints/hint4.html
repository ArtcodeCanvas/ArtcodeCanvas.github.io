第四课：用条件语句打造互动性 - 提示

任务回顾：根据天气状况绘制不同的景象。

如何使用条件语句：

  • 检查 isClearSky 变量是否为 true 来决定绘制明亮的天空还是阴暗的天空。
  • 在函数 drawSky() 中使用 if...else 语句来实现这一逻辑。
  • 考虑使用 isDaytime 变量来决定是否绘制太阳或月亮。
  • 使用逻辑运算符 && 来组合条件，例如 isDaytime && isClearSky 检查是否是白天且晴朗。

提示：仔细阅读任务要求，理解每个逻辑运算符的作用和如何应用它们来实现多条件的判断。

// 示例：使用嵌套 if 语句
if (isDaytime) {
    drawSun(); // 绘制太阳
    if (isClearSky) {
        drawHappyChild(); // 绘制快乐的小朋友
    }
} else {
    drawMoonAndStar(); // 绘制月亮和星星
    if (!isClearSky) {
        drawSadChild(); // 绘制不快乐的小朋友
    }
}

使用这些提示修改代码，并根据不同的天气状况来绘制相应的景象。
